## jQuery 로 복잡한 프로젝트에서 적용이 어려운 이유
---
* 느려졌다.
* 콘텍스트 메뉴얼 라이브러리 (Context Manual Library)
* 짧은 jQuery 로도 html 의 전체 DOM 을 스켄 할 수 있다.
* React 가 왜 빠르냐, DOM 에 바로 적용시키지 않고, 실제 브라우져에 모든 테스트를 끝내고 적용시킨다.
* 가상 DOM, 일종의 캐싱
* js -> 가상 DOM -> view
* 웹브라우저 감옥에서 벗어나기
